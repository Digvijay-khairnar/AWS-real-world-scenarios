# AWS Load Balancer Scenario-Based Interview Questions (1 Year Experience)

## Question 1

### Scenario

Users cannot access your application through the Load Balancer. What will you check first?

### Answer

- Check whether the Load Balancer is Active.
- Verify EC2 instances are running.
- Check Target Group health.
- Verify Security Groups.
- Check Listener configuration.
- Verify DNS settings.

---

## Question 2

### Scenario

Your Target Group shows instances as Unhealthy. Why?

### Answer

- Incorrect Health Check Path
- Application not running
- Wrong Port
- Security Group blocking traffic
- Health Check Timeout
