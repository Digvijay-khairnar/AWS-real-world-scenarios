# AWS Load Balancer Scenario-Based Interview Questions (2 Years Experience)

## Question 16

### Scenario

Your application receives five times more traffic during peak hours. How would you handle it?

### Answer

- Attach Auto Scaling Group
- Configure Scaling Policies
- Use CloudWatch Metrics
- Automatically add/remove EC2 instances

## Question 17

### Scenario

Your application has multiple microservices. Which Load Balancer will you use?

### Answer

Use Application Load Balancer because it supports:

- Path-Based Routing
- Host-Based Routing

Example

/users → User Service

/orders → Order Service

/payment → Payment Service
